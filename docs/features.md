# Detected LaTeX Commands

Here is the list of LaTeX commands or environment that FLaP detects and adjusts.

 * File inclusions directives:
    * `\input`
    * `\include`
    * `\includeonly`
 * Graphics inclusion directives:
    * `\includegraphics`
    * `\graphicspath`
 * Beamer directives:
    * `\begin{overpic}`
    * `\endinput`
